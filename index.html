Pajama

A programming language for 2D sprite games.

Overview

Pajama Lang is a translated language for the purpose of creating 2D games. It is almost a proper subset of C#’s syntax, where the eventual goal is to actually be able to compile and debug directly in C#. Currently, it gets translated into JavaScript and Python (with PyGame). Once exported, the output files should work out of the box and require no post-export tweaks. It is ready to post or share and is guaranteed to behave identically on both platforms.

Pajama is Object Oriented and type safe. Variables must be declared before they are used and can only be assigned compatible types. Typo’d variable usages are compile time errors, unlike either of the platforms it exports to.

Current State (as of Aug 1, 2013)

Pajama was written in a weekend. Last weekend, actually. It's very unstable at the moment but is surprisingly usable. If you're feeling adventurous, feel free to help test it.
